Getting Started with Railo

Thank You for Downloading Railo - an ultra fast Open Source CFML Engine.

Starting Railo

To start the Railo service, simply run the start script for your environment, e.g. start.bat for Windows, or start for Linux or Mac OSX.

By default Railo runs on TCP port 8888, so once you run it on your local machine, for example, you can test that it is working by visiting http://127.0.0.1:8888/

Running Your Own Code

The default folder that runs as the web root is {this-folder}/webapps/www

To test out your own code simply place it in that folder and browse to the URL above.

Railo Administrator

The Railo Server Administrator is available at the virtual folder /railo-context/admin/server.cfm so once you have Railo running on this machine, for example, you can visit it at http://127.0.0.1:8888/railo-context/admin/server.cfm

The Railo Web Administrator, for a specific website, is available at the virtual folder /railo-context/admin/web.cfm and is based on the host name of the url, so for example http://site1/railo-context/admin/web.cfm will allow you to set specific settings for site1, while http://site2/railo-context/admin/web.cfm will allow you to set specific settings for site2
